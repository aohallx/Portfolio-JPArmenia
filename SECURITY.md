# Security Policy

## Supported Versions

This repository is actively maintained on the `main` branch.

## Reporting a Vulnerability

If you discover a security issue, please report it privately by email:

- `johnparmenia@gmail.com`

Please include:

- A clear description of the issue
- Steps to reproduce
- Any proof-of-concept details (if available)
- Potential impact

I will acknowledge receipt as quickly as possible and work on a fix before public disclosure.

## Disclosure Guidelines

- Please do not publicly disclose vulnerabilities until a fix is available.
- After remediation, coordinated disclosure is welcome.
